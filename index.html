Premières étapes avec Git

- mkdir hello : pour creér le fichier
- cd hello
- git init
- git status
- Ajout du fichier index.html
- Pour suivre le projet : git add(+ nom du fichier ici index.html)
- git commit -m "message correspondant au commit"
- git log = historique de tous les commits créés
git log --online git log --online -p index.html = tous les commit du fichier - Créer un fichier contact.html
- git add contact.html
- git add une deuxième fois pr qu'il soit en statut staged
- git status
- git commit -m "ajout page contact.html"
- modification du fichier

- git + une lettre + tabulation = donne les possibilités avec la première lettre saisie

- git diff (si on précise le fichier :index.html) = montre tous les changements du fichiers, (si on ne précise rien montre ts les fichiers)

- git add * pour faire tous les fichiers d'un coup 2bdoieyhep

revenir en arrière

- git checkout index.html = retire tout pour revenir au derneir git add
