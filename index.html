Premières étapes avec Git

- mkdir hello : pour creér le fichier
- cd hello
- git init
- git status
- Ajout du fichier index.html
- Pour suivre le projet : git add(+ nom du fichier ici index.html)
- git commit -m "message correspondant au commit"
- git log = historique de tous les commits créés
- Créer un fichier contact.html
- git add contact.html
- git status
- git commit -m "ajout page contact.html"
- modification du fichier
